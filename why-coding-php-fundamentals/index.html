			Parsed Strings
		

		
    
			
  • Interprets special characters
    • 
		


"\n" › newline
"\"\$x\"" › "$x"

		
    
			
  • Variables are expanded
    • 
		


"$x" › value of x
"{$x[1]}" › value of x[1]
"{$x['date']}" › value of x['date']
"{$obj->month[12]->name}" › December
"{$x["date"]}" › Parse Error

		


		

		

			Unparsed Strings
		

		
    
			
  • Doesn't interpret special characters
			


'\n' › \n
'\'\$x\'' › '\$x'

		
    
			
  • Variables aren't expanded
		


'$x' › $x
'{$x[1]}' › {$x[1]}
'{$x["date"]}' › {$x["date"]}
'{$obj->month[12]->name}' › {$obj->month[12]->name}
'{$x['date']}' › Parse Error
