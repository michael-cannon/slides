    • close PHP code using ?>
  • Everything outside <?php & ?> is ignored
  • Separate PHP instructions via ; or ?>
  • Commenting
http://www.php.net/manual/en/language.basic-syntax.php

Booleans 

$bool = true;
$bool = false;
  • Simplest type
  • Expresses a truth value
  • Either TRUE or FALSE
  • CaSe-iNsEnSiTiVe (use lowercase)
http://www.php.net/manual/en/language.types.boolean.php

Integers 

$int = 1234; // decimal number
$int = -123; // a negative number
$int = 0123; // octal number (83 decimal), 0 precede
$int = 0x1A; // hexadecimal number (26 decimal), 0x precede
$int = 0b101; // hexadecimal number (5 decimal), 0b precede
  • Whole number
    • Decimal (base 10)
    • Hexadecimal (base 16)
    • Octal (base 8)
    • Binary (base 2)
  • Positive + or negative -
http://www.php.net/manual/en/language.types.integer.php

Floating Point Numbers 

$float = 1.234; 
$float = 1.2e3; 
$float = 7E-10;
  • Also known as floats, doubles, or real numbers
  • Positive + or negative -
  • Beware of NaN
http://www.php.net/manual/en/language.types.float.php

Strings 

echo 'this is a simple string';

$str = <<<EOD
Example of string
spanning multiple lines
using heredoc syntax.
EOD;
  • Series of characters
  • Can be as large as 2GB
  • Parsed or unparsed
http://www.php.net/manual/en/language.types.string.php

Unparsed Strings 

'\n' › \n
'\'\$x\'' › '\$x'
'$x' › $x
'{$x[1]}' › {$x[1]}
'{$x["date"]}' › {$x["date"]}
'{$obj->month[12]->name}' › {$obj->month[12]->name}
'{\$x["date"]}' › {\$x["date"]}
'{$x['date']}' › Parse Error
  • Single quoted ' (simplest)
  • Nowdoc syntax (PHP 5.3.0) <<<'EOD' & EOD;
  • Does not interpret special characters
  • Variables are not expanded

Parsed Strings 

"\n" › newline
"\"\$x\"" › "$x"
"$x" › value of x
"{$x[1]}" › value of x[1]
"{$x['date']}" › value of x['date']
"{$obj->month[12]->name}" › December
"{\$x['date']}" › {$x['date']}
"{$x["date"]}" › Parse Error
  • Double quoted "
  • Heredoc syntax <<<EOD & EOD;
  • Interprets special characters
  • Variables are expanded

Arrays

  • An ordered map that associates values to keys
  • Also known as list, hash table, dictionary, collection, stack, or queue
  • Array types
    • Simple
    • Associative
    • Multi-dimensional
http://www.php.net/manual/en/language.types.array.php

Simple Arrays 

$array = array( 10, 20, 30 );
$array = array( 'I', 'am', 'TYPO3', );
$array = [ 'are', 'you?', ]; // as of PHP 5.4
  • Comma-separated values
  • Last array element comma is optional
  • PHP 5.4 short array syntax [] vs. array()
http://www.php.net/manual/en/language.types.array.php

Associative Arrays 

$array = array( 'foo' => 'bar', 'bar' => 'foo', );
$array = array( $key1 => $value1, $key2 => $value2, );
$array = array( 10 => 'foo', 20 => 'bar', ]; // as of PHP 5.4
  • Comma-separated key => value pairs
  • Key is an integer, string or constant
http://www.php.net/manual/en/language.types.array.php

Multi-dimensional Arrays 

$array = array(
	'fruits'  => array ( 'a' => 'apple', 'b' => 'banana', ),
	'numbers' => array ( 1, 2, 3, 4, 5, 6 ),
	'holes'   => array ( 'first', 5 => 'second', 'third' ),
);

echo $array['fruits']['a']; // apple
echo $array['numbers'][2]; // 3
echo $array['holes'][5]; // second
unset( $array['holes'][0] ); // removes first

$array['apple']['green'] = 'good'; // new multi-dimensional array
  • A simple or associative array of arrays
http://www.php.net/manual/en/language.types.array.php

Type Juggling 

$foo = '0'; // $foo is string (ASCII 48)
$foo += 2; // $foo is now an integer (2)
$foo = $foo + 1.3; // $foo is now a float (3.3)
$foo = 5 + '10 Little Piggies'; // $foo is integer (15)
$foo = 5 + '15 Small Pigs'; // $foo is integer (20)
PHP does not support type definition… type is determined by context…
http://www.php.net/manual/en/language.types.type-juggling.php

Simple Math 

$a = 1; // Sets value of $a to 1
$b = 3; // Sets $b to 2
$c = $a + $b; // Sets $c to the sum of $a and $b
$d = $a - $b; // Subtracts $b from $a and stores it in $d
$e = $a * $b; // $e becomes $a multiplied by $b
$f = $a / $b; // Sets $f to $a divided by $b
$a++; // Increments the value of $a
$b--; // Subtracts 1 from the value of $b
$a += $b; // sets $a = $a + $b
$a -= $b; // sets $a = $a - $b
$b *= $c; // sets $b = $b * $c
$b /= $c; // sets $b = $b / $c
$g = 3 * 3 % 5; // (3 * 3) % 5 = 4
$h = true ? 0 : true ? 1 : 2; // (true ? 0 : true) ? 1 : 2 = 2
http://www.higherpass.com/php/Tutorials/Php-Fundamentals/3/
http://www.php.net/manual/en/language.operators.arithmetic.php
http://www.php.net/manual/en/language.operators.precedence.php
